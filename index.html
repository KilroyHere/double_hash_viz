Probing: Linear Probing (cluster walk)  vs  Double Hashing (cluster skip)

QUERY 1 / 5 ← → · space
✗ Linear Probing — step = 1, walks through cluster
✓ Double Hashing — unique step per key, skips cluster
Linear Probing — Probe
h = hash1(k) mod T
while table[h] ≠ k and table[h] ≠ EMPTY:
  h = h + 1
▲ step=1 → all probes pile into the same cluster
Double Hashing — Probe
h = hash1(k) mod T
step = hash2(k) mod (T−1) + 1
while table[h] ≠ k and table[h] ≠ EMPTY:
  h = h + step
▲ unique stride per key → jumps past clusters
